# Simple Password Manager

A simple, local Android application to manage your passwords securely using Room database and Jetpack Compose.

## Features

- **Store Passwords**: Save account names, usernames, and passwords.
- **View List**: Easily browse all your saved credentials in a clean list.
- **Add New Credentials**: Quick entry via a simple dialog.
- **Delete Entries**: Remove credentials you no longer need.
- **Local Storage**: All data is stored locally on your device using Room Database.

## Tech Stack

- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Database**: Room
- **Architecture**: MVVM (Model-View-ViewModel)
- **Dependency Management**: Gradle (Kotlin DSL)

## Project Structure

- `data/`: Contains the Room database configuration, DAOs, and Entity models.
- `ui/`: Contains the Composable functions and UI themes.
- `MainActivity.kt`: The main entry point of the application.

## Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Build and run the app on an emulator or a physical device.
